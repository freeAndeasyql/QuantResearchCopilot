# Phase 1 阶段日志

## 日期

2026-07-01

## 阶段主题

Vue3 + FastAPI 股票行情基础功能联调。

## 本阶段完成了什么

1. 创建 Vue3 + TypeScript 前端项目。
2. 创建 FastAPI 后端项目。
3. 实现 `/api/health` 健康检查接口。
4. 配置 CORS，完成第一次前后端联调。
5. 实现股票 Mock 数据接口。
6. 实现股票搜索、行业筛选、后端分页。
7. 实现股票详情接口。
8. 实现股票历史价格接口。
9. 前端使用 ECharts 展示价格走势图。
10. 统一后端成功响应和错误响应结构。
11. 编写接口文档 `docs/api.md`。

## 本阶段理解了什么

1. 前端通过 Axios 调用 FastAPI 接口。
2. FastAPI 通过路由函数返回 JSON 数据。
3. CORS 用来解决前后端端口不同导致的跨域问题。
4. 后端分页比前端分页更接近真实业务场景。
5. 统一响应结构可以降低前端适配成本。
6. Mock 数据适合在没有数据库前快速推进功能。
7. ECharts 需要有明确宽高的 DOM 容器才能正常渲染图表。

## 当前遗留问题

1. 股票数据仍然是 Mock 数据，尚未接入数据库。
2. 历史价格数据只有少量股票。
3. 页面样式还比较基础。
4. 暂未实现真实 K 线图和收益率计算。
5. GitHub push 受网络影响，部分提交可能仅保存在本地。
