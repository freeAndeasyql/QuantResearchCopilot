# 项目目标

## 1. 项目名称

Quant Research Copilot

## 2. 项目定位

这是一个面向投资研究和量化分析的 AI 应用项目。

目标不是只学习 Python、SQL 或前端，而是做出一个完整的作品集项目：

Vue3 前端页面 → FastAPI 后端接口 → 数据库查询 → 量化指标计算 → 图表展示 → AI 投资分析。

## 3. 核心功能

1. 股票行情数据查询
2. 因子分析
3. 策略回测
4. 收益风险指标展示
5. 研报上传与 RAG 问答
6. AI 投资研究助手
7. 登录权限与 API 安全
8. 系统日志与操作记录

## 4. 技术栈

Frontend: Vue3 + Vite + Pinia + ECharts  
Backend: FastAPI + Python  
Database: PostgreSQL / DuckDB  
AI: RAG + Tool Calling + Structured Output  
Security: Auth + API Security + LLM Security  

## 5. 六个月目标

第 1 月：完成行情数据清洗、入库、查询和图表展示。  
第 2 月：完成 FastAPI + Vue3 前后端交互和 CSV 上传。  
第 3 月：完成因子分析、策略回测和收益风险图表。  
第 4 月：完成研报 RAG、AI 投资问答和工具调用。  
第 5 月：完成登录权限、API 安全、LLM 安全和 AI 风险评估。  
第 6 月：完成部署文档、演示视频、README 和作品集材料。
